Chassis & Modules

Set how many OPX1000 chassis your setup has, then click a slot to insert a MW-FEM or LF-FEM module.

Keyboard: Tab into the grid, ← / → move between slots, ↑ / ↓ or M / L set the FEM, Del clears, Enter opens the menu.

{# OPX+ / Octave are hidden until the generator can actually build them: deriveLines hard-pins the readout to kind:"mw_fem", and an OPX+/Octave spec fails deep in the allocator (NotEnoughChannelsException) or, for mixed OPX1000+OPX+, in build_instruments. Until deriveLines emits opx/octave channel kinds, the wizard only offers OPX1000 MW/LF — the path verified end-to-end. Handlers stay bound for when it lands. #}

This wizard builds OPX1000 setups with MW-FEM / LF-FEM modules. OPX+ and Octave support is coming soon.

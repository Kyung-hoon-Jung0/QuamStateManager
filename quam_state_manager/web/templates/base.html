Welcome to QUAM State Manager

Inspect, edit, generate, and track your QUAM chip state — a working copy always sits between you and the live instrument, so nothing reaches the hardware by accident.

Get started from the left panel: State Load opens a chip's quam_state, Dataset Load adds an experiment folder.

{# Core mental model — kept always-visible; everything else keys off these three. #}
Live chip — the state.json / wiring.json your instrument & Qualibrate actually read.
Working state — your private editable copy; edits here never touch the instrument until you Apply to live.
Snapshot — an immutable, timestamped archive (in State History) you can roll back to.
{# Feature tour — collapsible so the page stays compact by default but each area carries a real description. Left chevron rotates on open; Pico's right-floated summary::after is suppressed in .welcome CSS. #}

What you can do — click a topic to expand

🔎 Explore & edit the state

Find any value and change it safely.

  • Browse in Explorer, or a Qubit / Pair inspector
  • Search any property — results in <1 ms
  • Edit inline → the change lands in the working state
  • Many fields at once? Live State Edit gives a grid
〰️ Pulses & waveforms

Manage pulses with a live preview.

  • Open Pulses — every pulse in one table
  • Edit a field → the waveform preview redraws live
  • Create / rename / duplicate / delete (with “used-by” safety)
  • Config Viewer → the waveforms after generate_config()
🛠️ Generate a fresh config

Build a new chip config from scratch.

  • Run the Generate Config wizard
  • Enter instruments, LO plan, qubit specs
  • Check the topology + wiring previews
  • Preview generate_config() → save state.json + wiring.json
📊 Chip Status dashboards

The whole chip’s health at a glance.

  • Topology · Overview · Distributions — layout & summaries
  • Coherence · Fidelity · Frequencies — heatmap views
  • Gate (2Q) · Calibration — per-pair readouts
⇄ Compare across time & devices

Spot exactly what changed.

  • Same chip, two times → drift over time
  • Two devices → side-by-side (auto grid-map)
  • Set a tolerance → hide noise, show real diffs
🕘 History, snapshots & rollback

Nothing is lost — roll back anytime.

  • 🔖 bookmark → pin a tagged archive before risky edits
  • Param History → per-parameter trends
  • State History → stage or restore any snapshot
📁 Datasets & interactive plots

From measurement to calibrated value.

  • Pick an experiment run in the sidebar
  • Open its data in the interactive viewer
  • Click a fit point → stage the value (confirm popup, never silent)
🔗 Sync-Qualibrate & operations

Run alongside Qualibrate and keep watch.

  • Sync-Qualibrate → Qualibrate side-by-side
  • Diagnostics badge → always-on health
  • Scheduler → drive the run queue
  • Fit Audit → triage calibration fits
